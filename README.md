## V1.0(2016/5/27)
 * 注册、登录
 * 主界面tab选项卡（分联系人、群、消息）
 * 根据QQ号搜索添加好友及群
 * 支持在线聊天、离线消息缓存、群聊
 * 支持发送QQ表情
 * 支持拖动窗口到顶部隐藏主面板、图标闪烁效果和语音提示消息效果
